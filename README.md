# Oral Cancer Detection App

AI-powered oral cancer detection using Deep Learning and Flutter.

## Project Overview
This app uses MobileNetV2 deep learning model to detect oral cancer from images with 94%+ accuracy.

## Features
- MobileNetV2 Transfer Learning Model
- TFLite on-device inference (9.9MB model)
- Flutter Web App
- Flask REST API
- 94%+ detection accuracy

## Tech Stack
- AI/ML: Python, TensorFlow, TFLite
- Backend: Flask, Flask-CORS
- Frontend: Flutter, Dart
- Dataset: Oral Cancer Images (Normal + Cancer)
